# NEVA — ГРАНИЦЫ СИСТЕМЫ
Версия: 0.1 | Дата: 2026-05-30

## ЧТО NEVA ДЕЛАЕТ
- Хранит контекст всех агентов (Claude, ChatGPT, будущие)
- Обеспечивает преемственность между чатами
- Синхронизирует состояние проектов (ARKA, SARJ, другие)
- Даёт агентам единую память независимо от канала доступа
- Управляет потоками документов по регламенту v6
- Следит за лимитами токенов и ресурсов

## ЧТО NEVA НЕ ДЕЛАЕТ
- Не принимает решения
- Не заменяет governance/*
- Не дублирует Git
- Не хранит RAW медданные (это SARJ)

## КТО ПОЛЬЗУЕТСЯ NEVA
- Claude Desktop (через MCP + arka-memory)
- API агенты (OpenRouter / GitHub Actions)
- Серж напрямую (чтение статуса)
- Любой новый AI при сбое (через docs/README_EMERGENCY.md)

## ОГРАНИЧЕНИЯ
- FREE-FIRST: все компоненты бесплатны
- THERMAL-SAFE: работает на M1 8GB без перегрева
- OFFLINE-SAFE: базовые функции без интернета

## ДИСПЕТЧЕР — ЕДИНСТВЕННАЯ ТОЧКА ВХОДА
Все потоки документов и данных — только через dispatcher/
Никто не пишет в память напрямую
