## ИСПРАВЛЕНИЕ

ChatGPT Круг 6: heartbeat обновляется до check_triggers.
Если триггер завис (kill_graceful, osascript) — Medic не узнает.

ПРИНЯТО. Финальный порядок цикла:

```python
while True:
    current_state = check_all_10()        # мониторинг
    write_atomic('inspector_status.json') # запись состояния
    check_triggers(current_state)         # действия (kill, restart)
    Path('state/inspector_heartbeat').touch()  # пульс ПОСЛЕ всего
    time.sleep(30)
```

Теперь если любой шаг завис — heartbeat не обновится — Medic обнаружит через 90с.
