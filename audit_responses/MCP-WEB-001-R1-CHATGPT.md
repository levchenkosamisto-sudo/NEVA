1. Поддерживает ли веб-версия ChatGPT подключение к кастомному MCP-серверу?

Да, поддерживает, но не бесплатно для всех пользователей.

На июнь 2026 MCP доступен через функцию «Коннекторы» и пользовательские интеграции в экосистеме ChatGPT. Для подключения собственного MCP-сервера требуется поддерживаемый тариф (обычно Plus/Pro/Team/Enterprise, в зависимости от региона и текущей политики OpenAI). Настройка выполняется через интерфейс ChatGPT или через клиентские приложения OpenAI, где добавляется MCP-источник данных.

Для пользователя бесплатного веб-интерфейса ChatGPT прямое подключение произвольного локального MCP-сервера обычно недоступно.

2. Есть ли бесплатный способ подключить ChatGPT к внешней системе через MCP?

Да, но не через бесплатную веб-версию ChatGPT.

Варианты:

Использовать OpenAI API и самостоятельно реализовать MCP-клиент в своей программе.

Использовать локальные среды разработки с поддержкой MCP (например, IDE-агенты и десктоп-клиенты), которые подключаются к вашему MCP-серверу.

Использовать собственный оркестратор NEVA, который обращается к OpenAI API и параллельно работает с MCP-серверами.

Для архитектуры NEVA наиболее практичный бесплатный путь выглядит так:

NEVA
 ├─ MCP серверы
 ├─ Git память
 ├─ Оркестратор
 └─ OpenAI API

Тогда MCP обслуживает сама NEVA, а ChatGPT используется через API без необходимости подключать MCP непосредственно к веб-интерфейсу.
