Подключение кастомного MCP к ChatGPT (web-версия)

Да. В 2025–2026 OpenAI добавила поддержку коннекторов и MCP для ChatGPT, однако доступ зависит от типа аккаунта и постепенно расширяется. Официальная документация: OpenAI MCP Documentation

Общая схема:

Поднимается MCP-сервер (локальный или удалённый).

В ChatGPT добавляется MCP-коннектор.

После подключения ChatGPT получает доступ к инструментам и данным MCP-сервера.

На практике поддержка доступна не на всех тарифах и может отличаться между Plus, Pro, Team и Enterprise. Актуальный статус лучше проверять на официальной странице OpenAI: OpenAI Help Center

Бесплатный способ подключения через MCP

Для полноценного MCP-доступа к моделям OpenAI обычно требуется:

аккаунт OpenAI;

API-ключ;

оплачиваемое использование API (по мере расхода токенов).

Полностью бесплатного официального MCP-доступа к моделям OpenAI через API сейчас нет.

Если задача — построить систему NEVA с MCP без затрат, обычно используют:

Ollama
 + локальные модели;

Open WebUI
;

Claude Desktop MCP
;

бесплатные API-провайдеры (Cerebras, Groq и др.) через собственный MCP-шлюз.

Для архитектуры NEVA на Mac M1 наиболее реалистичный бесплатный вариант сейчас:

NEVA → MCP → Ollama / Cerebras / Groq
