## СИСТЕМА
NEVA — агентная система на Mac M1 8GB RAM. Python 3.12. Без Docker. ARM64.
Директор: Серж. Архитектор: Claude.
Всё бесплатно и локально.

## КОМАНДЫ (использовать только эти, не угадывать)
```bash
./scripts/start.sh       # запуск NEVA
./scripts/test.sh        # запуск тестов
./scripts/index.sh       # переиндексация памяти
ruff check src/          # проверка стиля кода
```

## КРАСНАЯ ЗОНА (без явного разрешения Директора не трогать)
- governance/decisions/   — решения Директора
- src/core/neva_auth.py  — авторизация
- src/memory/schema.sql  — схема базы данных
- .env                   — ключи и токены

Нарушение красной зоны = блокировка коммита через pre-commit хук.

## ПРАВИЛА РАБОТЫ С КОДОМ
- Менять только то что указано в задаче
- Не переписывать рабочий код без измеримой причины
- Каждое изменение покрывать тестом
- После изменения: ./scripts/test.sh — все тесты должны пройти
- Формат коммита: [TASK-XXX] заголовок / что изменено / N PASS
